# 游历年轴 `GamingEpochs`

## [by 序碑工作室 Refillsoft](https://github.com/XuBeiStudio)

该项目是用于展示与订阅各平台游戏的发售时间以及日程更新，也可以用于查看近期发售的游戏详情如`发行商`、`开发商`、`会员免费信息`等。
